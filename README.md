
# DAM2.PJ6 Android Multimedia and Services

- `M01_AudioActivity.java`: Play audio from an Activity with basic controls (play/stop).
- `M02_AudioIntentService.java` / `M02_AudioServiceActivity.java`: Play audio in the background using a Service/IntentService and control it from the Activity.
- `M03_ExecutorServiceActivity.java`: Use an ExecutorService to run background tasks and synchronize results with the UI.
- `M04_Notificacions_SnackBar.java`: Display notifications and use a SnackBar for short user messages.
- `M05_CanvasBasic.java`: Introduction to Canvas for drawing basic shapes in a View.
- `M06_CanvasPersonalitzat.java` / `M06_VistaPropia.java`: Create a custom View that draws with Canvas and manages the drawing lifecycle.
- `M07_CanvasPath.java` / `M07_PathDit.java` / `M07_VistaPaint.java`: Work with Path and Paint for freehand paths and advanced paint configurations.
- `M08_AlarmaActivity.java` / `M08_AlarmaReceiver.java`: Set up AlarmManager with a BroadcastReceiver to perform scheduled actions.
- `M09_Propietats_dinamiques.java`: Modify view properties at runtime (dynamic properties) to prepare for animations.
- `M10_AnimacioBasica.java`: Apply basic view animations (translation, scale, rotation, alpha).
- `M11_AnimacioRequadre.java`: Animate a frame or UI elements to create more complex visual effects.
- `M12_CameraIntentActivity.java`: Use an intent to open the camera and capture a photo (ACTION_IMAGE_CAPTURE).
- `M13_CameraActivity.java`: Access the camera more directly (API) to capture and manage images.
- `M14_ImatgeDeSDActivity.java`: Read and load images from external storage (SD/external storage).
- `M15_OpenGLES20Activity.java` and related classes: Introduction to OpenGL ES 2.0 with GLSurfaceView and rendering simple shapes.
- `M16_LecturaXMLActivity.java` / `M16_LecturaXMLUtility.java`: Read and parse XML files to extract structured data.
- `M17_FragmentsActivity.java` / `M17_LlistaFragmentActivity.java` / `M17_DetallFragmentActivity.java`: Use fragments for modular UI (list + detail, multi-pane).
- `M18_Artista.java` / `M18_Firebase_Artistes.java` / `M18_Llista.java`: Integrate with Firebase to fetch and display a list of artists.
- `M19_AnimacioDescriptor.java`: Define reusable animation descriptors or configuration objects.
- `M20_AnimacioSprite.java`: Sprite animation (frame-by-frame) to display animated sequences.
- `M21_SumaServeiActivity.java` / `M21_SumaService.java`: Implement a Service that performs computations (sum) in the background and communicates the result to the Activity.
